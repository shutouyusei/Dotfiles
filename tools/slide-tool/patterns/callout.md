# Callout

Box with left accent bar and tinted background. Use for key messages, questions, or warnings.

## Default (blue)

```html
<div class="callout">

**Key point:** This is important information.

</div>
```

## Question for advisor (red)

```html
<div class="callout red">

**Question for advisor:** Which approach should we take?

</div>
```

## Color variants

- `class="callout"` — blue
- `class="callout green"` — green
- `class="callout red"` — red
- `class="callout orange"` — orange
