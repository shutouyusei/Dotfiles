# Description List

Colored keyword on top, indented gray description below. Use for listing items with short explanations (e.g., research trends, features, pros/cons).

## Basic (blue keywords)

```html
<div class="desc-list">
<div class="term">Keyword 1</div>
<div class="desc">Short description of this item</div>
<div class="term">Keyword 2</div>
<div class="desc">Short description of this item</div>
<div class="term">Keyword 3</div>
<div class="desc">Short description of this item</div>
</div>
```

## With color variants

```html
<div class="desc-list">
<div class="term green">Positive item</div>
<div class="desc">Description</div>
<div class="term red">Negative item</div>
<div class="desc">Description</div>
<div class="term orange">Open question</div>
<div class="desc">Description</div>
</div>
```

## Color variants

- `class="term"` — blue (default)
- `class="term green"` — green
- `class="term red"` — red
- `class="term orange"` — orange
