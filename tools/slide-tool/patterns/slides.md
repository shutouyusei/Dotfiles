# Slide Types

Special slide layouts using `<!-- _class: ... -->` directives.

## Title slide

```markdown
<!-- _class: title -->
<!-- _paginate: false -->

# Presentation Title

## Subtitle

Author — Date
```

## Section divider

```markdown
<!-- _class: divider -->

# Section Name

Optional description
```

## End slide

```markdown
<!-- _class: end -->
<!-- _paginate: false -->

# Thank You

Questions?
```

## Content slide (default)

```markdown
<!-- header: Section Name -->

# Slide Title

Content here...
```
